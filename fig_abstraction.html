Levels of Abstraction in Computing

From Human Ideas to Machine Instructions

More Abstract (Human-Friendly)

Natural Language

"Give me R code to plot sales"

Most Abstract

Using everyday language to instruct AI. The computer interprets intent and context, requiring no technical knowledge.

High-Level Languages

plot(data$sales)

Human-Readable Code

Languages like Python, R, or Java use English-like syntax. A compiler or interpreter translates this into lower-level code.

Assembly Language

MOV AX, 0x4C00

Symbolic Machine Code

A low-level language that uses mnemonics to represent machine instructions. It's specific to a computer's architecture.

Machine Code

B8 00 4C CD 21

Hardware Instructions

The raw instructions executed directly by the CPU, often shown in hexadecimal for readability. This is the output of a compiler.

Binary Code

10111000 01001100 ...

Least Abstract

The fundamental language of computers. Everything ultimately becomes sequences of 1s and 0s, representing on/off electrical signals.

Less Abstract (Machine-Friendly)
