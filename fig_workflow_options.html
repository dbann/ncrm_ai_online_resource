How to Use LLMs for Coding

A breakdown of common workflows and tools.

LLMs Suggest Code
(User Executes)

Autocomplete

User writes partial code → LLM suggests completion → User accepts/rejects.

IDEs with AI features e.g., VS Code, RStudio, Positron

Chatbot Request

User requests code → Copy to IDE/stats program → User executes.

IDEs with AI features e.g., VS Code, RStudio, Positron Stats applications e.g., Stata, SPSS.

LLMs Suggest + Execute Code
(Automated)

Built-in Code Interpreters

User uploads data → LLM generates and executes code.

ChatGPT, Claude, DeepSeek, etc. LLM data analysis wrappers

IDE Agent Modes

Iteratively generates and executes code within the local development environment.

Cursor, Windsurf, VS Code

Command Line Interface

CLI apps with contextual knowledge of local files, codebooks, and data structures.

Claude Code, Gemini CLI, Qwen Code

Data Security Warning

Be careful with sensitive data (e.g., survey data not permitted on cloud servers). Consider using local LLMs for sensitive projects.
